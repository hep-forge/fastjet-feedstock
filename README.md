# fastjet-feedstock

[![Build AMD64](https://github.com/hep-forge/fastjet-feedstock/actions/workflows/autoupload.amd64.yml/badge.svg)](https://github.com/hep-forge/fastjet-feedstock/actions/workflows/autoupload.amd64.yml)
[![Build ARM64](https://github.com/hep-forge/fastjet-feedstock/actions/workflows/autoupload.arm64.yml/badge.svg)](https://github.com/hep-forge/fastjet-feedstock/actions/workflows/autoupload.arm64.yml)

Feedstock for [fastjet](https://fastjet.fr) — part of [hep-forge](https://anaconda.org/hep-forge).

A software package for jet finding in pp and e+e− collisions. 

## Install

```bash
conda install -c hep-forge -c conda-forge fastjet
```
